<h1>Projeto How VI - CupWatcher</h1>
-> Projeto acadêmico Univali.

Para iniciar o projeto execute os comandos: <br/>
<h3>Iniciar projeto com NPM</h3>
<code>npm -i</code>
<br/>
<h3>Start no projeto</h3>
<code>npm run initServer</code>
